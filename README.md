# zaehlerstand
Zählerstand Manager ist eine mit KI generierte simple App um Zählerstände zu ermitteln.
Es gibt auch eine Export und Backup Funktion.
## Wie funktioniert es?
Über den [Link](https://mischa-petschenig.github.io/zaehlerstand/) kommt man direkt zur Webapp.
Sie lässt sich völlig offline verwenden in dem man sie als Wepapp installiert, oder das Repo runterladet und auf dem Desktop eine Verknüpfung
auf die index.html einrichtet.

Die Daten werden nur im Browser gespeichert das bedeutet wenn man Browserdaten löscht ist alles weg.
Backup!
